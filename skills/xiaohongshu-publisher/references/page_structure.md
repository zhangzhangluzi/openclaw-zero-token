# 小红书创作平台页面结构参考

最后更新：2026-02-19

## 首页 (creator.xiaohongshu.com)

### 主要元素

- **侧边栏导航**：左侧包含“发布笔记”、“笔记管理”、“数据看板”等
- **顶部栏**：用户头像、账号名称
- **内容区**：默认显示发布选项

### 发布选项

在首页内容区顶部有三个选项：
- 上传视频
- 上传图文
- 写长文

**关键选择器**：
- 上传图文按钮：通常是一个包含“上传图文”文字的div，在快照中ref可能为 `e107`

---

## 图文发布页面 (/publish/publish?source=official)

### 页面布局

顶部是标题输入区，中间是正文编辑区，底部是发布按钮。

### 标题输入框

位于页面顶部，通常是一个单行输入框。
- placeholder："填写标题，可能会有更多赞哦～"
- 类型：input 或 textarea

### 正文编辑区

位于标题下方，是一个多行文本区域。
- placeholder："分享你的生活态度"
- 类型：textarea

### 图片上传区

位于正文下方，有两种状态：

**未上传时**：
- 一个大的上传区域，显示“拖拽视频到此或点击上传”
- 内部有一个隐藏的 file input

**已上传图片后**：
- 显示图片缩略图
- 每个图片右上角有删除按钮
- 可以继续添加图片

### 发布按钮

位于页面底部或右下角，通常是一个红色按钮。
- 文本："发布" 或 "发布笔记"
- 状态：有内容时可点击，无内容时置灰

## 快照元素参考（基于2026-02-19抓取）

### 首页快照片段
```
- generic [ref=e107] [cursor=pointer]: 上传图文
- generic [ref=e109] [cursor=pointer]: 上传图文
```

### 发布页可能元素
- 标题输入框：可能带有 `placeholder="填写标题"` 的 input
- 正文区：textarea 元素
- 发布按钮：`button` 包含文本 "发布"
- 图片上传区：`input[type="file"]` 元素（通常是隐藏的）

## 注意事项

1. 小红书可能会进行A/B测试，不同账号看到的界面可能有细微差异
2. 发布页面的 URL 可能是 `https://creator.xiaohongshu.com/publish/publish?source=official`
3. 如果直接访问发布页未登录，会跳转到登录页，需要先登录
4. 图片上传后可能需要等待处理完成才能发布（显示上传进度）
5. 发布成功后通常会跳转到笔记管理页或显示成功提示

## 维护建议

当技能失效时，建议按以下步骤排查：
1. 手动打开 `https://creator.xiaohongshu.com` 观察界面变化
2. 使用浏览器开发者工具检查关键元素的 selector
3. 更新本文件中的 selector 信息
4. 同步更新 `publish_xiaohongshu.py` 脚本中的等待条件
