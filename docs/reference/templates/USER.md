# USER.md - User Profile

This file contains information about the user for personalized interactions.

## Purpose

The USER.md file helps the agent understand the user's preferences, context, and needs.

## Format

```markdown
# User Profile

## About Me
*(Information about yourself)*

## Preferences
- *(Your preferences)*

## Context
- *(Relevant context for the agent)*
```

## Example

```markdown
# User Profile

## About Me
Software developer working on AI projects.

## Preferences
- Prefer concise explanations
- Use code examples when possible
- Focus on practical solutions

## Context
- Working timezone: UTC+8
- Primary languages: English, Chinese
- Tech stack: TypeScript, Python, React
```
